# vue-storybook

Разбираюсь в функционале Storybook

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run the storybook
```
npm run storybook
```

### Run tests
```
npm test
```
